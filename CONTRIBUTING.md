# Contributing

Contributions that improve documentation, fix errors, or add tested configurations are welcome.

## How to Contribute

1. Fork the repository
2. Create a branch: `git checkout -b fix/description`
3. Make your changes
4. Submit a pull request with a clear description

## What's Helpful

- **Hardware compatibility reports** — tested on a different Pi model or HAT revision?
- **Documentation corrections** — found a step that's out of date or unclear?
- **Additional application guides** — Jellyfin, Pi-hole, Grafana, etc.
- **Troubleshooting additions** — solved an issue not covered in the docs?

## Guidelines

- Keep commands tested and working
- Never commit credentials, IPs, or personal information
- Use placeholder values: `<YOUR_USERNAME>`, `<YOUR_NAS_IP>`, `<DRIVE-UUID>`
- One logical change per pull request
